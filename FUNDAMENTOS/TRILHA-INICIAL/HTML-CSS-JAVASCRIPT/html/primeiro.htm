O que significa a sigla HTML?



Vem da palavra em inglês, HyperText Markup Language.
Que em português significa: Linguagem de Marcação de Texto.

A tag "< br>" é usada para quebrar linha e pode ser chamada das seguintes formas:

  • Elemento;
  • Tag;
  • Marcação;


    • A ideia de HyperText, vem do inicio da internet, quando existiam os "blogs", onde eram produzidos textos enormes e eram criados LINKs para outros textos.
    Hoje em dia além de textos, temos: jogos, videos, audio, aplicações em tempo real (real time) e existem diversas aplicações que utilizam o HTML por debaixo, pois, a Linguagem evoluiu para atender as demandas que não foram pensadas.
    Temos outras Tecnologias que também evoluíram com o tempo como:

  • CSS;
    • Usado para estilizar as páginas (veremos mais a frente).

  • Redes;
    • Onde temos o protocolo HTTP (utilizado para acessar as páginas WEB) que significa HyperText Transference Protocol ou em português: Protocolo de Transferência de Hyper Texto.

    Anatomia da TAG





    Agora iremos ver as possibilidades em relação a uma TAG:
    • #1 - Vimos que uma TAG se inicia com um simbolo de menor que " <" depois o nome da TAG "br" e finaliza com o simbolo de maior que ">" essa é uma das possibilidades de se usar uma TAG, lembrando que não pode haver ESPAÇO entre o simbolo "<" e o nome da TAG.
      Um exemplo prático seria na construção de um parágrafo representado pela TAG de abertura < p> e a de fechamento < /p> e supondo que no meio do parágrafo tenha uma palavra em negrito, será usado a TAG de inicio < b> e a de fechamento < /b>.

      Lorem ipsum dolor sit amet consectetur adipisicing elit. Itaque, ab saepe? Error odio corrupti dolores voluptatibus in incidunt ipsum quasi. Possimus eos earum corporis totam placeat praesentium impedit eius ab.

    • #2 - No segundo caso vimos que a TAG foi aberta e fechada ao mesmo tempo utilizando o "/>" após o nome da TAG.
    • #3 - No terceiro exemplo, vimos a TAG com dois atributos um do Tipo numérico "x=123" e outro do Tipo String "y='Olá'"
    • #4 - Outra forma é criando uma TAG com corpo tendo um INICIO e um FINAL "< xyz>" e "< /xyz>" e no meio colocando o conteúdo referente aquela TAG.
      Com isso, é possível colocar uma TAG dentro da outra por Exemplo:

      < xyz>
      < abc />
      < def />
      < ghi />
      < /xyz>
    • #5 - No quinto exemplo temos a TAG com abertura, atributos, corpo e fechamento.

      < xyz a="123" b="teste">
      < abc />
      < def />
      < ghi />
      < /xyz>
