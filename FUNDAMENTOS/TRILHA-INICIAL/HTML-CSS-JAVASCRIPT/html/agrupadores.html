Agrupadores

Divs: são containers (blocos) que ocupam uma linha inteira, por padrão, assume a altura de seu conteúdo e toda a largura da página.

Span: são espaços em linha que por padrão, assumem a altura e a largura de seu conteúdo.

Os mesmo através de CSS podem ser configurados para assumirem diversos comportamentos.
