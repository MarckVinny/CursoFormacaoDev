## Definindo Estrutura de Pastas do Projeto

Inicialmente iremos definir a estrutura de pastas, onde iremos criar da seguinte forma:

```zsh
.
|-- ANOTAÇÕES.md
|-- css
|   |-- estilo.css
|   `-- menu.css
|-- evento
|   `-- index.html
|-- index.html
|-- modificar
|   `-- index.html
`-- selecionar
    `-- index.html
```
